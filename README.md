# web-ai-translator
Hệ thống AI Agent, mỗi agent phụ trách xử lý một phần trong quy trình dịch thuật: phân tích cấu trúc tài liệu đầu vào, trích xuất, chia nhỏ văn bản, sử dụng LLM để dịch có kiểm soát, tái tạo lại tài liệu đầu ra và kiểm tra kết quả. Xây dựng bằng Python, kết hợp công nghệ browser automation, thư viện xử lý tài liệu, công cụ render và mô hình AI.
